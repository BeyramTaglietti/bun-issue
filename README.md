# nsp-rda-new

To install dependencies:

```bash
bun install
```

To install Turborepo:

```bash
bun install -g turbo
```

To run:

```bash
bun run dev
```

To build:

```bash
bun run build
```

This project was created using `bun init` in bun v1.3.0. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
